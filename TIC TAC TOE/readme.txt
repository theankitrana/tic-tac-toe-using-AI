 ******************* TIC TAC TOE USING AI***************
    This Game Is Created Using JavaScript, html, CSS, Canvas

                **Overall Description**
    A 1D Array called "gameData" to store the players moves.
    Draw the board on the canvas.
    add Event listener to the CANVAS.
    Determine which SPACE on THE BOARD the player has  clicked on.
    Update the gameData array. (if the space clicked by the player is empty)
    Draw the player's move on the board (canvas).
    check if this player wins. if wins we show the game over message. and stop the game.
    If it doesn't win, we check for a tie game, if it's a tie we show the game over message. and stop the game.
    If there is no winner and it's not a tie game, we give the turn to other player.
